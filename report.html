编程练习3：定义衡量标准¶

如果不能对模型的训练和测试的表现进行量化地评估，我们就很难衡量模型的好坏。通常我们会定义一些衡量标准，这些标准可以通过对某些误差或者拟合程度的计算来得到。在这个项目中，你将通过运算决定系数 R2 来量化模型的表现。模型的决定系数是回归分析中十分常用的统计信息，经常被当作衡量模型预测能力好坏的标准。

R2的数值范围从0至1，表示目标变量的预测值和实际值之间的相关程度平方的百分比。一个模型的R2 值为0还不如直接用平均值来预测效果好；而一个R2 值为1的模型则可以对目标变量进行完美的预测。从0至1之间的数值，则表示该模型中目标变量中有百分之多少能够用特征来解释。模型也可能出现负值的R2，这种情况下模型所做预测有时会比直接计算目标变量的平均值差很多。

在下方代码的 performance_metric 函数中，你要实现：

  • 使用 sklearn.metrics 中的 r2_score 来计算 y_true 和 y_predict的R2值，作为对其表现的评判。
  • 将他们的表现评分储存到score变量中。

或

  • (可选) 不使用任何外部库，参考决定系数的定义进行计算，这也可以帮助你更好的理解决定系数在什么情况下等于0或等于1。

from sklearn.metrics import r2_score
def performance_metric(y_true, y_predict):
    """计算并返回预测值相比于预测值的分数"""
    
    score = r2_score(y_true, y_predict)

    return score

def performance_metric2(y_true, y_predict):
    """计算并返回预测值相比于预测值的分数"""
    sum_te = 0
    sum_pe = 0
    mean = np.mean(y_true)
    
    for i in range(len(y_true)):
        sum_te += y_true[i] - mean
        sum_pe += y_predict[i] - mean
    
    score = sum_pe / sum_te

    return score

问题 8 - 回答：¶

  • K折交叉验证。将数据集分为K个样本，进行K次分组训练验证模型，将其中1份作为验证数据集，K-1份作为训练数据集，确保每个样本都充当了一次验证数据集。将K次交叉验证的平均结果作为模型的评估。
  • GridSearchCV结合网格搜索，对不同的参数组合进行K折交叉验证，分别得到对应的平均评估分数，取其中最高的参数组合作为最优的训练模型参数。
  • 'cvresults'属性返回一个字典，记录每一组参数的评估结果，比如分数、时间等。
  • 如果不是用交叉验证，模型的评估很难具有完备性和全面性，很难找到最优的参数组合。交叉验证通过尽可能将所有的数据集作为验证数据集参与进模型的评估，保证了评估过程的完整和全面。

from sklearn.model_selection import KFold
from sklearn.tree import DecisionTreeRegressor
from sklearn.metrics import make_scorer
from sklearn.model_selection import GridSearchCV

def fit_model(X, y):
    """ 基于输入数据 [X,y]，利于网格搜索找到最优的决策树模型"""
    
    cross_validator = KFold(n_splits=10)
    
    regressor = DecisionTreeRegressor(random_state=0)

    params = {'max_depth': range(1, 11)}

    scoring_fnc = make_scorer(performance_metric)

    grid = GridSearchCV(regressor, params, scoring=scoring_fnc, cv=cross_validator)

    # 基于输入数据 [X,y]，进行网格搜索
    grid = grid.fit(X, y)

    # 返回网格搜索后的最优模型
    return grid.best_estimator_

    estimator.fit(X1+X2, Y1+Y2)
    prices1 = estimator.predict(X3)
    scores1 = scoring(Y3, X3)
    estimator.fit(X1+X3, Y1+Y3)
    prices2 = estimator.predict(X2)
    scores2 = scoring(Y2, X2)
    estimator.fit(X3+X2, Y3+Y2)
    prices3 = estimator.predict(X1)
    scores3 = scoring(Y1, X1)
    
    scores = (score1 + score2 + score3) / 3
    
    return scores


def fit_model2(X, y):
    """ 基于输入数据 [X,y]，利于网格搜索找到最优的决策树模型"""
    score = 0
    for i in range(2,11):
        regressor1 = DecisionTreeRegressor(max_depth=i, random_state=0)
        score1 = cross_val_score(regressor1, X, y, scoring = performance_metric, cv=3)
        if score1 > score:
            score = score1
            regressor = regressor1
    
    #最优交叉验证分数对应的最优模型
    best_estimator = regressor
    
    return best_estimator

    grid = grid.fit(X, y)
